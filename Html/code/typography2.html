Heading text tag :-

Code :- 

<!DOCTYPE html>
<html lang="en">
<head>
    <title>Typography text tags</title>
    <link rel="stylesheet" href="codeCSS.css">
</head>
<body>
    <h1> heading one </h1>
    <h3> heading third </h3>
    <h2> heading second </h2>
    <h5> heading fifth </h5>
    <h4> heading fourth </h4>
    <h6> heading sixth </h6>
</body>
</html>

Output :-

heading first

heading second

heading third

heading fourth

heading fifth
heading sixth
